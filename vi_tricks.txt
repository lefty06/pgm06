cit = to delete all content within a html tag (insert mode)
ci" = to delete all content within "contents" (insert mode)
. = to repeat all command types in insert mode (escape mode)
set number|set nu =to show line number,to cancel set nonu (escape mode)
set relativenumber|set rnu =to show relative line number, to cancel set nornu (escape mode)
p = to paste before cursor, P after cursor (insert mode)
H = to get the cursor at the top of the page (escape mode)
L = to get the cursor at the bottom of the page (escape mode)
f{character} = to search and move the cursor to the first occurence of the letter on the right and F{character} on the left (escape mode)
